A. Perangkat Keras - Hardware

1. Perangkat Masukan (Input Devices)

Peranti masukan adalah perangkat yang mengirimkan data ke komputer untuk diolah, seperti:

  • Mouse wireless Bluetooth: menggerakkan kursor dan memilih menu.

    • Mouse atau tetikus digunakan untuk menggerakkan dan mengatur posisi kursor di layar komputer. Pengoperasian mouse dapat dengan beberapa cara. Tunjuk (Point), posisikan: Mouse akan menunjuk ke suatu objek antarmuka di layar komputer. Klik kiri (Left Click): Menekan tombol mouse sekali dan segera melepaskannya. Klik ganda (Double Click): Menekan tombol mouse dua kali berurutan dengan cepat tanpa menggeser mouse. Klik kanan (Right Click): Klik kanan dapat dilakukan satu kali untuk menampilkan menu tertentu. Seret (Drag): Digunakan untuk memindahkan suatu objek antarmuka seperti gambar, icon, teks, dan sebagainya. Caranya dengan menunjuk objek yang akan dipindah sambil menekan tombol kiri mouse, lalu geser mouse sesuai yang dikehendaki. Setelah sampai pada bagian yang dikehendaki tombol mouse dilepas

  • QWERTY Keyboard.: mengetik teks dan simbol.

    • Keyboard QWERTY, yaitu keyboard dengan tata letak paling populer yang didasarkan pada enam huruf pertama pada papan ketik mesin tik. Desain QWERTY dibuat agar huruf yang paling umum tidak akan membuat mesin tik mekanis “macet”, atau berhenti bekerja.

  • Photo Scanner: memindai dokumen ke format digital.

    • Scanner digunakan untuk memindai dan menyalin Data dari kertas berisi grafik, gambar, foto, atau tulisan menggunakan alat pembaca optical data reader. Cara kerja scanner adalah dengan menempatkan sebuah obyek di atas kaca scanner, yang kemudian akan dibaca oleh optical data reader

  • Joystick: mengontrol permainan.

    • Joystick digunakan sebagai pelengkap untuk memainkan permainan/game video yang menggunakan lebih dari satu tombol. Joystick digunakan untuk menggerakan kursor pada layar permainan. Elemen-elemen joystick adalah: 1. Stick/tuas, 2. Base/alas, 3. Trigger/Pemicu, 4. Tombol ekstra, 5. Sakelar autofire, 6. Sakelar throttle, 7. Sakelar tapi (POV hat), 8. Cangkir penyedot

  • Microphone: merekam suara.

    • Microphone adalah peranti yang mengonversi suara menjadi sinyal elektrik. Peranti ini diguna kan untuk memasukkan suara ke komputer. Suara tersebut dapat direkam sebagai perintah untuk komputer, atau diteruskan melalui media komunikasi antarkomputer.

  • Barcode Reader: membaca kode batang produk.

    • Barcode reader adalah pemindai optis yang dapat membaca barcode yang tercetak, meng ubah kode bar tersebut menjadi data elektrik dan mengirimkannya ke komputer dengan format data yang sederhana.

2. Perangkat Pemroses

Data atau perintah yang diterima dari peranti masukan selanjutnya akan diproses oleh komputer. Pemroses ini terdiri atas banyak komponen, yaitu seperti berikut.

  • CPU: pusat kendali dan perhitungan komputer (CU & ALU).

    • CPU adalah otak komputer yang bertugas menjalankan semua instruksi, seperti operasi aritmatika, logika, pengendalian, dan input/output dasar. Komponen utama CPU: Control Unit (CU): Mengatur dan mengendalikan jalannya instruksi. Arithmetic Logic Unit (ALU): Menjalankan perhitungan dan operasi logika. Register: Menyimpan data sementara yang sedang diproses. CPU terpasang di mainboard (motherboard) dan terhubung dengan perangkat lain seperti memori utama, alat input/output, dan GPU (untuk grafis). Kecepatan CPU diukur dalam satuan Hertz (Hz), umumnya dalam skala MHz atau GHz.

  • RAM: menyimpan data sementara.

    • RAM (Random Access Memory) adalah memori cepat yang menyimpan data sementara saat komputer memproses informasi. Data dari penyimpanan sekunder akan dimuat ke RAM agar bisa diolah CPU lebih cepat. file tersebut dimuat dan dimanipulasi di RAM. RAM memungkinkan akses data secara acak, tanpa urutan tertentu. Karena menggunakan arus listrik dan transistor, RAM memiliki latensi rendah, tetapi data akan hilang saat listrik mati.

  • ROM: menyimpan instruksi permanen BIOS.

    • ROM adalah memori pada komputer yang digunakan untuk menyimpan instruksi yang ditulis oleh produsen komputer untuk proses booting, instruksi yang disimpan oleh ROM disebut firmware. Data yang pernah ditulis di ROM tidak dapat dihapus. Instruksi ROM dijalankan secara otomatis saat komputer menyala. Pada komputer, ROM bisa berisi sistem operasi kecil yang disebut Basic Input Output System (BIOS)

3. Perangkat Keluaran (Output Devices)

Peranti keluaran adalah peralatan yang menyajikan informasi dari komputer yang bisa berupa tampilan di layar monitor, gambar tercetak, suara yang dihasilkan oleh speaker, dan lainnya. Peranti keluaran di antaranya adalah sebagai berikut.

  • Monitor: menampilkan informasi visual.

    • Layar monitor berfungsi untuk menampilkan hasil pengolahan data di komputer, seperti gambar, teks, angka, grafik, dan sebagainya. Ada beberapa jenis monitor yang biasa digunakan, seperti Cathode Ray Tube (CRT), Liquid Crystal Display (LCD), dan Light Emitting Diode (LED)

  • Printer: mencetak informasi ke media kertas.

    • Printer digunakan untuk mencetak gambar, foto, dokumen dalam bentuk media kertas. Ada beberapa jenis printer yang kini masih digunakan di antaranya printer dot matrix, inkjet, laserjet, dan plotter

  • Speaker: mengeluarkan suara dari komputer.

    • Alat untuk menghasilkan suara dari komputer, seperti musik, percakapan di film, dan efek suara lainnya dinamakan speaker

4. Perangkat Penyimpanan Sekunder

Adalah media penyimpanan data permanen yang menyimpan data meskipun listrik mati (non-volatile). Jenis-jenisnya:
1. Hard Disk Drive (HDD) Menggunakan piringan magnetik untuk menyimpan data. Kapasitas umum: 256 GB – 18 TB. Komponen utama: piringan logam, head baca-tulis, rangkaian elektronik. Data tidak hilang meskipun listrik mati.
2. Solid State Drive (SSD) Menggunakan flash memory tanpa komponen bergerak. Lebih cepat, tahan guncangan, dan tidak berisik dibanding HDD. Kapasitas hingga 100 TB (2019). Data tetap ada saat listrik mati.
3. Flash Drive (USB) Portable, menggunakan USB interface dan flash memory. Kapasitas bervariasi: 8 GB – 2 TB. Digunakan untuk menyimpan, mencadangkan, dan memindahkan data.
4. Compact Disc (CD) Cakram optik digital. Digunakan untuk audio atau data. Format: CD-ROM, CD-R, CD-RW, VCD, dll. Kapasitas umum: 700 MB.
5. Memory Card (SD Card) Digunakan pada kamera, smartphone, dan tablet. Ada fitur CPRM dan Write-Protect. Bentuk: SD, MiniSD, MicroSD (paling kecil).
6. Cloud Storage Penyimpanan data melalui internet ke server penyedia layanan. Data bisa diakses di mana saja, kapan saja dengan internet. Contoh layanan: Google Drive, Dropbox, pCloud. Keunggulan: akses fleksibel, berbagi mudah, dan keamanan terjamin.

5. Perangkat Lainnya

Selain komponen utama, ada peranti tambahan yang mendukung fungsi dan kinerja komputer. Beberapa di antaranya:
1. Motherboard (Mainboard) Merupakan papan sirkuit utama dalam komputer. Tempat menghubungkan dan memasang komponen penting seperti: Processor (CPU), RAM, Hard Disk / SSD,VGA Card, Sound Card, dll. Semua perangkat berkomunikasi melalui motherboard.
2. VGA Card (Video Graphics Array Card) Berfungsi untuk mengolah data grafis/video dari CPU. Hasil olahan ditampilkan di layar monitor. Penting untuk tampilan visual, seperti game, video editing, dan desain grafis.
3. Sound Car Mengubah sinyal digital dari komputer menjadi sinyal suara. Sinyal suara dikirim ke speaker atau headphone. Digunakan untuk mendengarkan musik, suara video, dan kegiatan audio lainnya.
4. GPU (Graphics Processing Unit) Merupakan prosesor khusus untuk menangani pemrosesan grafis dan gambar. Lebih efisien dan cepat daripada CPU dalam mengolah grafis karena struktur paralelnya.
Digunakan dalam:
Komputer pribadi (PC)
Laptop
Konsol game
Ponsel dan sistem tertanam (embedded system)
Bisa berupa:
kartu grafis terpisah (dedicated GPU)
GPU tertanam di motherboard atau CPU (integrated GPU)
